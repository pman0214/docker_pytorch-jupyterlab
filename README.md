# pytorch + Jupyterlab

## Overview

* Docker image based on `pytorch/pytorch`.
* Installs python libraries below:
  * seaborn
  * jupyter
  * jupyterlab (password: jupyter)
  * scikit-learn
  * imblearn
  * statsmodels
